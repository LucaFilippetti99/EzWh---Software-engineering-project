# EZ Warehouse

**Team project for the Software Engineering course** **Master's Degree in Computer Engineering at Politecnico di Torino (2025/26).**

EZ Warehouse is a warehouse management system supporting product catalog, inventory tracking, inbound/outbound orders, shipment status, and multi-role user management.

## Project Structure

```text
├── code/                   # System implementation
├── API.md                  # REST API specification
├── DesignDocument.md       # Architecture and design details
├── RequirementsDocument.md # Functional and non-functional requirements
├── OfficialRequirements.md # Course assignment specification
├── GUIPrototype.md         # User interface prototype
├── Estimation.md           # Effort estimation and planning
└── TimeSheet.md            # Team time tracking and activities
```
The project emphasizes analysis, design, and documentation following software engineering best practices.

## Features

* **Product catalog and inventory management**
* **Order processing** (inbound/outbound)
* **Shipment tracking**
* **Role-based user access** (admin, warehouse staff, operators)
* **REST API** for system integration

Detailed requirements and API endpoints are in the respective Markdown files.
